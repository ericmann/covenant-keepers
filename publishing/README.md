# Publishing Directory — Covenant Keepers Series

This directory contains publishing assets and metadata for the Covenant Keepers children's book series by Eric A Mann, published by Jumping Duck Media (an imprint of Displace Technologies, LLC).

## Contents

| File | Purpose |
|------|---------|
| `copyright-page.md` | Template copyright page for all books in the series |
| `series-metadata.md` | Series-level Amazon/KDP metadata |
| `amazon-keywords.md` | 7 keyword phrases per book (all 12 books) |

Book-specific front matter and back matter files live in each book's story directory:
- `stories/NN-book-slug/FRONT-MATTER.md`
- `stories/NN-book-slug/BACK-MATTER.md`

## KDP Upload Checklist

Use this checklist for each book when publishing through Amazon Kindle Direct Publishing.

### Pre-Upload Preparation

- [ ] Final manuscript proofread and approved
- [ ] Interior PDF formatted for 5.5" x 8.5" trim size
- [ ] All 8 illustrations placed and sized correctly in interior PDF
- [ ] Front matter (half title, title page, copyright, dedication, TOC) included in interior PDF
- [ ] Back matter (About the Author, About the Series, Coming Next, Reading Order) included in interior PDF
- [ ] Cover files ready: front cover, back cover, and spine (combined as single wrap PDF/image per KDP specs)
- [ ] Spine text sized appropriately for page count (KDP spine calculator)
- [ ] Back cover includes series branding, book description, barcode area

### KDP Account Setup (One-Time)

- [ ] KDP account created and verified
- [ ] Tax information submitted
- [ ] Bank account linked for royalty payments
- [ ] Publisher name set: Jumping Duck Media

### Metadata Entry (Per Book)

- [ ] Book title and subtitle entered
- [ ] Series name: Covenant Keepers
- [ ] Series volume number (1-12)
- [ ] Author: Eric A Mann
- [ ] Book description (from series-metadata.md, customized per book)
- [ ] 7 keyword phrases entered (from amazon-keywords.md)
- [ ] Amazon categories selected:
  1. Books > Children's Books > Action & Adventure
  2. Books > Children's Books > Literature & Fiction > Religious Fiction > Christian > Action & Adventure
  3. Books > Children's Books > Literature & Fiction > Religious Fiction > Christian > Historical
- [ ] Age range: 8-12
- [ ] Grade range: 3-7
- [ ] Language: English
- [ ] Publication date set

### ISBN Assignment

- [ ] ISBN obtained (purchased or assigned through KDP)
- [ ] ISBN entered on copyright page in interior PDF
- [ ] ISBN entered in KDP metadata
- [ ] Separate ISBNs for paperback and ebook editions (if applicable)

### File Upload

- [ ] Interior PDF uploaded (manuscript)
- [ ] Cover PDF/image uploaded (front/back/spine wrap)
- [ ] KDP previewer used to check formatting
- [ ] Page count confirmed
- [ ] No formatting errors flagged

### Pricing

- [ ] List price set (consider series pricing consistency)
- [ ] Royalty option selected (35% or 70%)
- [ ] Territory rights configured (worldwide recommended)
- [ ] Kindle ebook price set (if applicable)

### Pre-Order Setup (If Applicable)

- [ ] Pre-order enabled with release date
- [ ] Placeholder or final interior file uploaded
- [ ] Final file uploaded at least 10 days before release
- [ ] Pre-order listing reviewed for accuracy

### Review and Proof Cycle

- [ ] Digital proof reviewed in KDP Previewer
- [ ] Physical proof copy ordered
- [ ] Physical proof reviewed for:
  - [ ] Cover color accuracy
  - [ ] Spine alignment
  - [ ] Interior margins and gutters
  - [ ] Illustration reproduction quality
  - [ ] Text readability at trim size
  - [ ] Page numbering
  - [ ] Front and back matter completeness
- [ ] Corrections made and re-uploaded (if needed)
- [ ] Second proof ordered (if corrections were significant)
- [ ] Final approval given

### Post-Publication

- [ ] Listing live and verified on Amazon
- [ ] Author Central profile updated
- [ ] Series page linked correctly
- [ ] A+ Content added (if eligible)
- [ ] Categories verified (may need to contact KDP support for third category)
