# Covenant Keepers

A 12-book faith-based adventure series for 3rd–5th grade readers (ages 8–10).

**Concept:** Magic Treehouse meets Superbook meets Indiana Jones. Contemporary kids travel through time into Biblical stories, seeking sacred relics tied to God's covenant promises.

## Production Notes

- **Text:** Human-authored outlines, AI-assisted drafting, human-edited final copy
- **Interior illustrations:** 8 per book, B&W line art, commissioned from a human illustrator using AI-generated concept art as references
- **Cover art:** Full color, human illustrator, series template design
- **Publishing:** Amazon KDP (ebook + paperback), priced at $4.99 / $12.99
- **Target:** ~100+ copies per title via organic/community channels (Gaarde Christian School, ministry networks)

## Getting Started

1. Read `SPEC.md` for the full project plan
2. Read `SERIES-BIBLE.md` for the story universe
3. Read `AGENTS.md` for the AI-assisted writing workflow
4. Pick a book in `stories/` and start with its `OUTLINE.md`
